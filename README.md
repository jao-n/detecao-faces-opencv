# Detecção de Faces com OpenCV

Sistema de detecção facial em imagens utilizando OpenCV e o classificador Haar Cascade. Projeto completo de visão computacional para identificação automática de rostos humanos.

## Sobre o Projeto

Este código detecta faces humanas em imagens, desenhando retângulos verdes ao redor de cada rosto encontrado. Ideal para aprendizado em visão computacional e processamento de imagens.

## Funcionalidades

- **Carregamento inteligente de imagens** (suporte a vários formatos)
- **Pré-processamento automático** (conversão para tons de cinza)
- **Detecção de múltiplas faces** em uma única imagem
- **Marcação visual** com retângulos coloridos
- **Otimização de parâmetros** para melhor precisão
- **Visualização profissional** com matplotlib
